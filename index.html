An Easier Way to Visualize Data

WAVED simplifies the process of creating interactive web demos by providing users with an intuitive graphical user interface that allows for loading, manipulating, and visualizing data. Select widgets, bind your data, and export JavaScript for a completed application that can easily be embedded on the web.
